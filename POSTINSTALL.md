# Installation complete

Astrozop: Daily Horoscope Notifications has been successfully installed in your Firebase project.

## Next steps

To start sending notifications, you must integrate the Astrozop helper code in your app. Find out more about the helper code [here](https://github.com/gamezop/astrozop-firebase-extension-demo-apps)

The helper code will:

- Compute each user's sun sign from the birth date you provide
- Assign users to the correct local-time delivery window across timezones
- Automatically subscribe each user to the appropriate Firebase Cloud Messaging topic

Without this step, no notifications will be delivered.
